# Banner广告

### 加载Banner广告

```text
AdManager.getInstance().loadBanner();
```

### 判断广告是否加载

```text
if (AdManager.getInstance().bannerIsLoad()) {
    AdManager.getInstance().showBanner();
}
```

### 显示Banner广告

```text
AdManager.getInstance().showBanner();
```

### 隐藏banner广告

```text
AdManager.getInstance().hideBanner();
```
